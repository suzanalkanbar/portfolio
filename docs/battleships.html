Battleships Game

Interactive JavaScript project

Introductie

Dit Battleship spel heb ik ontwikkeld tijdens de minor Script voor Designers. De opdracht richtte zich op het leren programmeren van interactieve ervaringen met HTML5 Canvas en JavaScript. Het spel bevat interactieve scheepsplaatsing, verschillende moeilijkheidsniveaus en dynamische animaties voor treffers en missers.
